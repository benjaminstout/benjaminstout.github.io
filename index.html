EVI Unified Dashboard

A Unified data dashboard for electric vine industries
papua, indonesia

BACKGROUND

The EVI Unified Dashboard is a web application I built for Electric Vine Industries (EVI), a sub‑company of Advancing Engineering Consultants. The project was started during my internship in Papua, Indonesia during the summer of 2015. The goal of the Unified Dashboard was to consolidate and display customized data views of EVI's solar plants.

Because I only had a few short weeks to work on the dashboard while in Indonesia, EVI offered me a 4‑month contract to finish the project remotely.

IMPLEMENTATION

The dashboard is written in HTML, CSS, JavScript/jQuery and PHP with a MySQL database. Data is received through FTP uploads and secure cross-domain API cURL requests. Because of the confidential nature of this project the source code is not available publicly, however I would be happy to accommodate requests for it as best I can. SixPence

AN INNOVATIVE FUNDRAISING STARTUP

BACKGROUND

SixPence is a fundraising website and mobile application built from the ground up to simplify the fundraising process, making it as easy as handing a friend your spare change. SixPence operates on a new model of fundraising—rounding up to the nearest dollar on all of your daily purchases and donating this spare change to a campaign  of your choice.

The idea behind SixPence was thought of by a good friend of mine, Christopher Haylett. We met together in the Fall of 2015 to discuss the idea and to pursue making it a reality. I heartily agreed to spearhead  its creation and in the late Winter of  2015, I organized and led a small team of students from Seattle Pacific University to begin its development.

IMPLEMENTATION

The website is built using the CakePHP MVC framework and is written primarily in PHP with HTML and CSS. It will use the Plaid Connect API for transaction tracking and stripe for payment processing. It features phpMyAdmin for easy database management, hosted at the db subdomain.

This website is still heavily a work in progress.

Third‑World ECG

An affordable 5‑lead electrocardiogram for hospitals of developing nations

BACKGROUND

For my Computer Engineering capstone project, I worked on a team of 4 other electrical and computer engineers to design and build an affordable, modular, and easily maintainable 5‑lead electrocardiogram (ECG). With help from a contact in Ethiopia, our ECG was built from the ground up to fulfill the demands of most third‑world hospitals.

IMPLEMENTATION

Our ECG was designed to incorporate easily‑replaceable and interchangeable components. Some of its key features include:

  • built‑in surge‑protecting power supply, using a single varistor
  • 3.5mm audio cable electrode leads with snap connectors
  • Raspberry Pi processor
  • Standard 7‑inch TFT LCD
  • Custom‑designed PCBs

The software for it is written entirely in Python and uses the spidev, h5py, numpy, and pyqtgraph libraries.
